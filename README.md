# Tour Planner
This project is part of the Fachhochschule Technikum Wien Software Engineering Software Engineering bachelours.

## Overview

### How 2 Start
0. start mockoon mock service, by importing the mockoon.json and starting the mock server
1. navigate into the folder frontend
   1. run **npm install**
   2. run **npm run dev**
   3. enjoy

### Documentation
- Frontend -> React
- Backend -> ASP.NET
